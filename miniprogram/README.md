# ToolHub 微信小程序端

这是个人实用功能网站的微信小程序版本，复用现有 Django API 和账号体系。

## 已实现功能

- 登录 / 注册
- 首页工具入口
- 个人仪表盘
- 笔记 CRUD
- 待办 CRUD
- 密码生成、复制、保存
- 二维码生成和历史记录
- 个人中心 / 退出登录

## 导入方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本仓库下的 `miniprogram` 文件夹。
4. AppID 可以先使用测试号或你自己的小程序 AppID。
5. 开发阶段可勾选“不校验合法域名、web-view、TLS 版本以及 HTTPS 证书”。

## 配置服务器地址

接口地址在：

```text
miniprogram/utils/config.js
```

默认配置：

```js
API_BASE_URL: "http://114.55.52.162"
```

正式发布小程序时，微信要求请求域名必须是 HTTPS，并且需要在微信公众平台后台配置 request 合法域名。
